Start an auction now in just 3 steps

1. Register as a seller

It's simple, completely anonymous, and free. We do not collect any data and we do not track you.

All you need to do is to scan a QR code using a supported Lightning wallet and you are registered as a seller. We know absolutely nothing about you.

2. Create the auction

Not only we don't know who you are, but we also don't know what you are selling. All we store is properties that we actually need to run the auction: a start date, and end date, a minimum bid, that sort of stuff.

3. Paste the code into your website

Whether you run Wordpress or just generate your site statically using tools like Jekyll, we've got you covered. We give you a short snippet you include on your listing's page.
